## 🌟 نظرة عامة

المنصة الرسمية للأستاذ الدكتور **محمد لبيب سالم** - أستاذ علم المناعة بجامعة طنطا ورئيس الجمعية المصرية لأبحاث السرطان.

منصة ذكاء اصطناعي متطورة تقدم استشارات فورية ومتخصصة في مجال علم المناعة وأبحاث السرطان، مدعومة بأحدث تقنيات الذكاء الاصطناعي.

## ✨ الميزات الرئيسية

| الميزة | الوصف |
|--------|-------|
| 🤖 **مساعد ذكي** | محادثة ذكية متخصصة في علم المناعة وأبحاث السرطان |
| 🔐 **نظام تسجيل دخول** | دعم تسجيل الدخول عبر Google أو البريد الإلكتروني |
| 💾 **حفظ المحادثات** | حفظ جميع المحادثات تلقائياً مع إمكانية الرجوع إليها |
| 📱 **متجاوب بالكامل** | يدعم جميع الأجهزة (جوال، جهاز لوحي، كمبيوتر) |
| 🌐 **دعم اللغات** | يدعم العربية والإنجليزية |
| 📞 **تواصل مباشر** | زر واتساب احترافي للتواصل المباشر مع الدكتور |

## 🛠️ التقنيات المستخدمة

- **HTML5** - هيكل الصفحة
- **CSS3** - التصميم والتنسيق
- **JavaScript (ES6+)** - المنطق والتفاعلات
- **Firebase** - قاعدة البيانات والمصادقة
- **Groq API** - نموذج الذكاء الاصطناعي Llama 3.3
